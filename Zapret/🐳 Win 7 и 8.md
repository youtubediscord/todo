## Консольные версии (без GUI, Win7)

Специальная консольная версия с bat файлами стратегий для Windows 7+. Не поддерживается Python GUI.

Перед каждой сменой стратегии (`bat`) файла следует ВРУЧНУЮ закрыть все предыдущие окна программы и открыть новое окно.

При любых проблемах запустите `stop.bat` файл, чтобы остановить программу.

При любых ошибках следует использовать релизную GUI версию как более стабильную.

Автозапуск поддерживается только вручную (через добавление в реестр или папку `Start Menu`)
